## High Resolution
ハイレゾは96kHz/24bitという高いサンプリング数と、高い解像度を誇ります。  

通常、YouTubeでは44kHz/16bitで音楽が再生されるので、及ばないのですが、15kHz/16bitの音源を44kHz/16bitに引き上げてみます。
